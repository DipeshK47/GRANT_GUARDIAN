# Canonical Data Model

The implementation currently targets these primary entities:

- Organization
- Programs
- Funders
- Funder Filings
- Opportunities
- Requirements
- Evidence Library
- Documents
- Budgets
- Draft Answers
- Tasks
- Reviews
- Submissions
- Reporting Calendar
- Lessons
- Agent Logs
- Auth Tokens
- Source Snapshots

These are represented both as local SQLite tables and planned Notion databases/views.
